Como fizemos a soma dos números?

Começamos incluindo os inputs, colocando ONCLICK"somar" para dar a ação de somar após clicar dentro do input:button.

Dentro de SCRIPT criamos 3 VARs e lincamos com os inputs criados. Nas VARs n1 e n2 usamos o Number(VAR.value) para fazermos a conversão de script para número, criamos a VAR s que é a soma de tn1 + tn2 e por ultimo imputamos o res.innerHTML e incluimos o testo que aparecerá quando o botão de soma for clicado mais o resultado das soma das variaveis.
