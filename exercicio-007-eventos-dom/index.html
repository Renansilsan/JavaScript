Eventos DOM

Fizemos este exemplo para mostrar algumas ações que podemos fazer com o DOM. Começamos incluindo um ID no elemento, isso dentro da parte HTML do documento, dentro do SCRIPT criamos o VAR e lincamos o VAR com o ID usando o window.document.getElementById, nas linhas abaixo incluímos 3 linhas de area.addEventListener, para identificar a ação que será feita. Na linha 1 incluímos o 'CLICK', para dar o comando que acontecerá algo após o click, e colocamos CLICAR para lincar com o FUNCTION. Nas outras functions foram feitos quase os mesmos processos, mudando a ação na segunda linha usamos o mouseenter, para indicar o que acontecerá quando o mouse entrar, e lincamos com a function usando o ENTRAR e na terceira linha incluímos a ação 'MOUSEOUT', ação para quando o mouse sair, e lincamos com a function usando o SAIR.

Depois abrimos o primeiro function com o nome de clicar( ), dentro do function colocamos area.innerText = 'Clicou' e area.style.background = 'red', para personalizar a ação e o mesmo foi feito nas outras functions.
