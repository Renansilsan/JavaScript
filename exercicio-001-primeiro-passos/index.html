Como criar uma caíxa de alerta interativa;

Começamos criando a TAG base SCRIPT, onde incluíremos os comandos em JAVASCRIPT, ultilizamos este exemplo para demonstrar a TAG WINDOW.ALERT, dentro de ('') incluíremos a mensagem que aparecerá na caíxa de alerta antes do site ser carregado, ultilizamos WINDOW.CONFIRM para que apareça ua caíxa de texto que tenha opções para selecionar, dentro de ('') incluir a mensagem desejada e incluimos o WINDOW.PROMPT para criar uma caíxa onde pode ser escrito texto dentro dela, escrever entre ('') a mensagem que vai aparecer na caíxa.
