Condição simples em JavaScript

var vel = 65
console.log(`A velocidade do seu carro é ${vel} Km/h`)
if (vel > 60) {
console.log(`Você ultrapassou a velocidade permitida, MULTADO!`)
}
console.log(`Dirija sempre usando cinto de segurança!`)

Exemplo de condição simples

Neste caso começamos criando a VAR vel, velocidade = 65, na inha seguinte incluimos o console.log e entre (``) colocamos e a VAR vel, seguimos na próxima linha com o IF (vel > 60) e na linha seguinte outro console.log, ou seja, caso a velocidade seja maior que 60 km/h aparecerá a frase, Você ultrapassou a velocidade permitida, MULTADO!, caso seja menor nada acontecerá, isso é uma condição simples.
