Operadores

Operadores aritimmétricos;

  • + - Adição;
  • - - Subtração;
  • * - Multiplicação;
  • / - Divisão;
  • % - Resto da visão;
  • ** - Potência.

Ordem de precedência das operações

1. ( ), 2. ** Potência, 3. * Multiplicação, / Divisão e % Resto da visão, 4. - Subtração, + Adição.

Operadores relacionais;

  • > - Maior que - 5 > 2;
  • < - Menor que - 3 < 4;
  • >= - Maior ou igual - 8 >= 8;
  • <= - Menor ou igual - 5 <= 7;
  • == - Igual - 5 == 5;
  • != - Diferente - 4 != 6.

Sem ordem de precedência, quem aparecer primeiro será operacionado.

Operadores lógicos;

! - Negação - Para preposições como "Não quero um carro que não seja vermelho, podendo ser de quaquer outra cor".

&& - Conjução - Para preposições como "Quero um carro que seja azul e um carro que seja preto, tem que ser obrigatiriamente 2 carros e que sejão desta cor".

Tabela de preposições com conjunção;

True + True = True

True + False = False

False + True = False

False + False = False

|| - Disjunção - Para preposições como "Quero um carro que seja verde ou prata, mas tem que ser uma destas 2 cores".

Tabela de preposições com disjunção;

True + True = True

True + False = True

False + True = True

False + False = False

Ordem de operação;

1 - Operadores aritimmétricos, 2 - Operadores relacionais, 3 - Operadores lógicos.

Exemplo de preposições

idade >= 15 && idade <= 17 - A idade está entre 15 e 17?

estado == 'RJ' || estado == 'SP' - O estado é RJ ou SP?

salário > 1500 && sexo != 'M' - O salário é acima de 1500 e não é homem?

Operador ternário

  • ?;
  • :.

Como funciona?

Onde colocaremos o uqe acontecerá quando as outras operações forem resolvidas, por exemplo

var media = 5.5
média > 7? 'APROVADO' : 'REPROVADO'

Conclusão - Caso a média seja acima de 7 aparecerá o que foi preenchido em ? e caso seja menor aparecerá o que foi preenchido no :.
